# Installation

## Stable release

To install Automated Video Generator, run this command in your terminal:

```sh
uv add automated-video-generator
```

Or if you prefer to use `pip`:

```sh
pip install automated-video-generator
```

## From source

The source files for Automated Video Generator can be downloaded from the [Github repo](https://github.com/harisongsouza/automated_video_generator).

You can either clone the public repository:

```sh
git clone git://github.com/harisongsouza/automated_video_generator
```

Or download the [tarball](https://github.com/harisongsouza/automated_video_generator/tarball/master):

```sh
curl -OJL https://github.com/harisongsouza/automated_video_generator/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd automated_video_generator
uv pip install .
```
